# Portfolio AI & Data Science - Indra Wijaya

Portfolio website untuk menampilkan proyek-proyek AI, Machine Learning, dan Data Science.

## Fitur

- ✅ Desain modern dan responsif
- ✅ Smooth scrolling navigation
- ✅ Mobile-friendly dengan hamburger menu
- ✅ Animasi fade-in pada scroll
- ✅ Sections lengkap: Hero, About, Skills, Projects, Metrics, Services, Contact
- ✅ Gradient backgrounds yang menarik
- ✅ Card-based layout untuk proyek dan keahlian

## Struktur File

```
portfolio/
├── index.html      # Halaman utama
├── styles.css      # Styling dan layout
├── script.js       # Interaktivitas dan animasi
└── README.md       # Dokumentasi
```

## Cara Menggunakan

1. **Buka file HTML langsung di browser:**
   ```bash
   # Buka index.html di browser favorit Anda
   ```

2. **Atau gunakan local server (disarankan):**
   ```bash
   # Menggunakan Python
   python -m http.server 8000
   
   # Menggunakan Node.js (http-server)
   npx http-server
   
   # Menggunakan PHP
   php -S localhost:8000
   ```

3. **Akses di browser:**
   ```
   http://localhost:8000
   ```

## Teknologi yang Digunakan

- HTML5
- CSS3 (dengan CSS Variables)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## Customization

### Mengubah Warna

Edit variabel CSS di `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --primary-dark: #4f46e5;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

### Menambah Konten

Edit file `index.html` untuk menambah atau mengubah konten portfolio.

### Mengubah Styling

Semua styling ada di file `styles.css`. Website ini menggunakan mobile-first approach dan fully responsive.
